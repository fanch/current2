Typical installation of NuTyX on a single partition

A dedicated single partition of 8 Gigabytes minimum, is needed for a full installation of NuTyX.

A minimum of 1 Gigabyte RAM is best suited. You have already burnt the ISO onto a CD.

If you prefer instead, prepare a NuTyX bootable USB flash-drive. Either option will work.

You also have a blank hard-drive.

Now we begin to use the following screen-shots, that will show you a typical installation on a single partition:

  1. Boot-up the USB Flash Drive /CD on which you copied/burned the ISO.

  2. Partition creation (optional)

    3. The partition creation is optional. If you already have an empty partition available you can directy go to the format of the partition.

  3. Format the partition (optional)

    4. The format of the partition is optional. If you do have an unformated or you want get rid of everything on it, then just format it. If you still want to keep your personal datas for example, you then need to remove the following directories: opt, bin, etc, root, run, sbin, usr and var. If you already have an empty formated partition then you can directly got to the installation of NuTyX

  4. Install NuTyX

    5. The installation of NuTyX doesn't need an internet. It takes just a few minutes

  5. Configure NuTyX

    6. Onces installed, NuTyX automatically start immediatly with its configuration without having to reboot.

    The configuration is very straight forward: network cards configuration, date and time configuration and modification of the default password of the administrator account root.

    Next, when prompted, use the default password nutyx to login for the first time. You will be required to change this password with a new password before you can proceed to the prompt.

  6. Giving the base commands a trial run-through.

    7. I suggest you to discover the base commands so that you can for example install a graphical interface.
