The base commands...

Give a try

To take full advantage of your newly installed NuTyX, here are the MUST commmands that you will have to use everyday. This article explain the essential commands and are NuTyX specifics. We strongly recommand to know them.

NuTyX do not have (yet) a graphical interface tool for administration purpose. You will then used the terminal as normal interface. If you are not yet used to it, it's not a big deal, we will explain you the commands to type.

Since NuTyX is a linux Operating system, it's important to respect the case. So it's important to respect ALL the caracters in lowercase.

Let's start to open a terminal. Ready ? Ok then let's go

In some case a command is made of one single word and without any option. But in most cases, a command can handle many options, we suggest you to use the help option to get an idea of all the options. Here, we will only check the most used one:

commandname help
or 
commandname --help
or 
commandname -h

the help give the possibility to printout an idea of all the availables options. They are also --help or -h as options to printout the help of the command.

Check if they are any updates available

The command cards sync will check the latest status of the available updates for your system.

The command cards diff will show a list of outdated packages

cards sync
cards diff
synchronizing /var/lib/pkg/saravane/desktop from http://..
/var/lib/pkg/saravane/desktop//MD5SUM
..

Updating a package

To update a package, the command cards install -u follow by the package will be used. It's important that the update is available:

cards install -u gcc

Search a package

To search a package, the command: 

cards search search_string

will return all the packages which have the search_string in the name or in the description. For example, if you want to know what's are the browser available

cards search browser

should return something like:

chromium           The open-source project behind Google Chrome, an attempt at creating a safer, faster, and more stable browser. Include PepperFlash and libpdf plugins
firefox            Standalone web browser from mozilla.org
...

Add a package

To add a package, the command cards install followed by the software name to install will be used:

cards install xfce4

The download of the application xfce4 starts immediately, once the package is downloads, it's dependencies will be downloads as well. When all done with the download, all the packages will be installed in the right order. In this example, when the commands is done you will have a working xfce4 desktop installed on your NuTyX.

Add a user

To add a user under NuTyX, it's very simple, the command:

nu

will add a user, it can be used in two forms with or without options, if used without options, they will be two question to answer:

User name: james
User description: James Bond 007

You should see that the user's name doen't contains certains characters neither space neither capital letters.

If you wish to use the command with the options, those hve to be given in the order user description then user name:

nu "James Bond 007" james

Alias

It's good to know that somes commands are actually alias which are simplify command. It's some how a shortcut to the original command. To know all the availables alias, just type in your terminal alias

Conclusion

As you can see the only two commands to knows are: 

nu
cards

This last command has a more detail explanation via it's help

cards help

I wish you a lot of fun with you newly installed NuTyX.
