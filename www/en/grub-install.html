GRUB Setting up

If you did'nt install GRUB yet ( by mistake or on purpose ), it's not too late. As you will see, it's very simple to install GRUB in a standalone way.

Historique

To be able to install GRUB

It's not necessary to have any NuTyX installed, you can indeed install GRUB for any other distribution.

Restart the PC and boot on the Iso

Once you arrive on the Main men, choose "Boot Configure the boot of the PC (optionnal)" like this:

It's now time to make ...

The selection of the partition

We choose the partition where the GRUB application will be installed. Please note that the destination partition doesnt have to be the first one which in this case will have the chance to choose another harddisk menu.

Grub is installed

It's now time to edit the grub.cfg file.

Be aware of the syntax of the adresssing of disks and partitions. It's a bit special.

With grub 2.00, the first harddisc start at 0 and the first partition start at 1, means in the following example:

set root=(hd0,5)

define the fift partition on the first harddisc

In the following example, we can see 3 entries, two of them are NuTyX ones, the first entry will start the NuTyX distribution locate on the /dev/sda2 partition. The second entry will start the maintenance of NuTyX locate on the /dev/sda1 partition and finally the last entry will start Windows 8 located on the /dev/sda3

# Begin grub.cfg
# By default boot the first menu entry.
set default 0
# Allow 5 seconds before booting the default.
set timeout 5
set color_highlight=blue/white
set color_normal=cyan/black
set gfxpayload=1024x768x16
menuentry "NuTyX  xfs sur /dev/sda2" {
 set root=(hd0,2)
 linux /boot/kernel root=/dev/sda2 ro quiet
 initrd /boot/initrd
}
menuentry "Maintenance de NuTyX" {
  set root=(hd0,1)
  linux /boot/kernel ro quiet
  initrd /boot/initrd
}
menuentry "Windows 8" {
 set root=(hd0,3)
 chainloader (hd0,3)+1
}
