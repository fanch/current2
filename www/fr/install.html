Installation classique de NuTyX sur une seule partition

Ces captures d'écrans vous montrent une installation classique sur une seule partition. Le disque dur est de 8 goctets et toute la place est réservée pour la première partition sur laquelle est installée NuTyX

La taille mémoire est de 1Goctet.

L'iso est transférée sur CD. Vous pouvez évidement utiliser une clé USB si votre PC le permet.

Le disque est totalement vierge et n'a jamais été formaté

  1. Démarrer la clé / le CD sur la(le)quel se trouve l'ISO

  2. Création de la partition (optionnel)

    3. La création de la partition n'est pas indispensable. Si vous avez déjà une partition disponible vous pouvez directement passer au formatage de la partition.

  3. Formatage de la partition (optionnel)

    4. Le formatage de la partition n'est à faire que si vous venez de la créer ou vous souhaitez effacer l'intégralité de son contenu. Si au contraire vous souhaitez conserver certains dossiers, vous pouvez alors supprimer uniquement les dossiers opt, bin, etc, root, run, sbin, usr et var qui se trouvent sur la partition de destination. Cette méthode permet de conserver vos données personnelles par exemple. Si la partition de destination est vierge et déjà formatée, vous pouvez directement passer à l'installation de NuTyX.

  4. Installation de NuTyX

    5. L'installation de NuTyX se fait sans connection réseau.

    Elle ne prend que quelques secondes

  5. Configuration de NuTyX

    6. Une fois installée, NuTyX se lance immédiatement sans avoir besoin de redémarrer la machine

    La configuration est très directe: réglage de la carte réseau, de la date, de l'heure et modification du mot de passe par défaut du compte administrateur root

    Le mot de passe par défaut est nutyx. Dès que vous aurez entrez ce mot de passe. On vous demandera de le changer immédiatement

  6. Première prise en main

    7. Je vous invite à prendre connaîssance des commandes de base pour que vous puissiez terminer votre installation avec une interface graphique par exemple.
