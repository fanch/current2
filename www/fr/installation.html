Installation

Assurez vous tout d'abord d'avoir un espace libre sur votre disque dur d'au moins 8Go ou mieux 15Go pour plus de confort.

Téléchargez la dernière version de NuTyX

La version actuelle de NuTyX s'appelle houaphan 8.0. Elle est disponible en 32 bits et en 64 bits. La version 64 bits n'est pas une multi lib mais une 64 bits pure. Les isos disponible permettent une installation sans avoir besoin de connection internet. Les isos étant le système de base. Elles permettent également de faire un minimum de maintenance sur la machine avant l'installation de NuTyX.

La taille des iso est +- 210 Mb chacune.

Les isos sont disponible à l'adresse: http://downloads.nutyx.org

Créez une clé bootable

Si vous êtes déjà sous GNU/Linux:

En root

A l'aide de fdisk, déterminez où se trouve votre clé USB:

fdisk -l

A l'aide de dd, créez votre clé bootable:

dd if=NuTyX_x86_64-houaphan-8.0.iso of=/dev/sdX

La valeur sdX est donnée par la commande:

fdisk -l

Si vous êtes sous Windows:

Utilisez votre outils de gravure favori pour graver l'iso sur un CD-RW (si possible) ...

Installation de l'amorçage de la machine

Cet article vous explique comment installer depuis zero l'amorçage de la machine

Pour les plus pressé et les plus callés, voici un résumé:

Si vous avez déjà grub 2 installé, il suffit de modifier votre grub.cfg selon:

menuentry "NuTyX sur /dev/sda1" {
    set root=(hd0,1)
    linux /boot/kernel root=/dev/sda1 ro quiet
    initrd /boot/initrd
    }
    
    menuentry "Windows 8" {
    set root=(hd0,3)
    chainloader (hd0,3)+1
    }
