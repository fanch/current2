Installation depuis le script

Quelque soit le moyen utilisé, l'installation de NuTyX se fait toujours par le biais d'un script. Il se faît que si vous le souhaitez vous pouvez directement l'exécuter depuis une NuTyX déjà installée ou depuis une autre distribution.
Sur les isos, le script d'installation est appelé par l'installeur semi-graphique. Il est possible d'installer NuTyX en utilisant directement ce même script depuis votre distribution actuelle. La procédure est assez simple. Cette méthode est très utile et peut être utilisée pour installer NuTyX sur une partition ou un dossier. Si vous installez NuTyX dans un dossier, vous pourrez "entrer" dans NuTyX en chroot par exemple. Le script est un moyen idéal dans le cas où:
  1. vous ne souhaitez pas réinstaller votre système d'exploitation
  2. vous souhaitez tester NuTyX dans un dossier ou sur une partition disponible.
  3. vous souhaitez construire vos propres paquets dans un environnement isolé aussi appelé "chroot".

Conditions à remplir:

Pour pouvoir installer NuTyX via le script, il faut s'assurer que les conditions suivantes soient remplies:
  1. La commande xz doit être disponible sur la distribution utilisée
  2. Le compte root doit être utilisé pour lancer le script
  3. L'interpréteur de commande doit être bash
  4. Si vous installez sur une partition, assurez-vous que la partition de destination est formatée en ext2, ext3, ext4, reiserfs ou btrfs
  5. Le script n'installera pas de gestionnaire d'amorçage (grub ou autre), c'est à vous de l'installer aprés ou avant le lancement du script
  6. L'installation de NuTyX se fait toujours sur une partition unique, si vous souhaitez séparer certains dossier /home, /var, etc, veuillez le faire après l'installation de NuTyX
  7. Le script ne crée pas d'utilisateur, seul le compte root est activé. Vous pouvez utiliser la commande nu une fois NuTyX installé.
  8. Pour installer une 64 bits veuilez utiliser une distribution 64 bits et pour une 32 bits, une 32 bits.
Il est vivement recommandé d'avoir l'application dialog d'installée afin de profiter des menus d'install ci-dessous. Si toutes ces conditions sont remplies, Téléchargez-le à l'adresse ci-dessous en utilisant le bouton de droite et "sauvegarder sous":

http://downloads.nutyx.org/install-sekong.5.ash

Veuillez vérifier sa somme md5:
http://downloads.nutyx.org/install-sekong.5.md5sum

Lancement du script

Une fois le script téléchargé, ouvrez un terminal en root et tapez: 
bash install-sekong.5.ash /dev/sda3 reiserfs fr_FR
L'installation commence directement. Veuillez ajuster la disposition du clavier au vôtre. Si vous installez depuis NuTyX, le clavier peut être omis, le script prendra celui défini par la NuTyX host. Si la disposition de votre clavier n'est pas reconnue par le script, vous pouvez utiliser la variable KEYMAP pour le spécifier. Exemple: 
KEYMAP=slovene bash install-sekong.5.ash /dev/sda3 reiserfs
Si vous avez votre propre dépot de paquets binaire, vous pouvez spécifier l'URL du dépot en utilisant la variable URL. Le même miroir sera utilisé une fois NuTyX installée. 
URL="http://192.168.1.1/NuTyX/" bash install-sekong.5.ash /dev/sda3 reiserfs fr_FR
Il faut savoir que si vous avez un média inséré dans le lecteur avec les dépots des paquets NuTyX (par exemple un liveCD ou une iso de base), le script identifiera le media et l'utilisera comme source de dépot. Cela peut vous paraître idiot, mais dans le cas de l'installation d'une chroot  c'est évidement très pratique car dans ce cas l'installation se fera en moins de 5 minutes. Si toutes les conditions sont remplies, il est temps de passer à l'installation de NuTyX

Installation de NuTyX

Une fois le script lancé, le premier menu apparaît, il s'agit de choisir un kernel. Plusieurs versions sont proposées en fonction de votre matériel. N'hésitez pas à essayer différents kernel en cas de problème
Notez que si vous le souhaitez, vous pouvez ne pas choisir de kernel, il sera alors possible de le compiler soi-même ou  même de l'installer une fois l'install terminée (en chroot)
Le menu suivant vous propose les différentes interfaces graphiques. Ici aussi, il est tout à fait possible de ne rien sélectionner dans ce cas vous installez une NuTyX de base.

Choix optionnel de linterface graphique

Une fois l'interface graphique choisie, vous pouvez affiner votre sélection en ajoutant les applications disponibles par groupe.
